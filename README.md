# SAFMC-D2
Ubuntu installation for Manifold/PC without Nvidia GPU: 
1. Download Ubuntu 16.04 from http://releases.ubuntu.com/16.04/ & Universal USB Installer from https://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/
2. Flash the Ubuntu image to your pendrive
3. Create a partition if you are dual booting
4. Restart your PC and select the pendrive with Ubuntu flashed inside
5. Select the “Install Ubuntu” option and install

Ubuntu installation for PC with Nvidia GPU: 
1. Download Ubuntu 16.04 from http://releases.ubuntu.com/16.04/ & Universal USB Installer from https://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/
2. Flash the Ubuntu image to your pendrive
3. Create a partition if you are dual booting
4. Restart your PC and select the pendrive with Ubuntu flashed inside
5. Select “Install Ubuntu” and press “e”, just before the “quiet splash — -”, write “nomodeset”, then press “F10” to run the installation
6. After reboot, open “Software and Updates” app to install install Nvidia driver

Follow the report for the software
